## Introduction

The dice command allows you to roll one or more dice.

| Argument | Description | Required |
|----------|-------------|----------|
| `type` | Define the types of dice you want to roll. | `true` |


``` bash
/dice type: D20

/dice type: D20, D20, D12, D8, D6
```
